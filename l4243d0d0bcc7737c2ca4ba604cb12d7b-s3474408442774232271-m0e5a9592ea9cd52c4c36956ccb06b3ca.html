User 模块

1. 一键登录接口
  • URL: http://121.4.51.120/ferry/user/login

  • 请求参数 loginToken

    A0000001至 A0000008 目前还没接入极光的接口 暂时用这8个值作为loginToken

  • 请求方式 POST

  • 返回结果
    { "msg": "登录成功", "code": 200, "data": { "userName": "fpga", "userId": "1", "avatar": "http://localhost:8080/ferry/1_.png", "sex": 1, "birth": 1607708827674, "phone": "13500000000" } }

    2. 获取验证码

  • URL: http://121.4.51.120/ferry/user/getVerificationCode

  • 请求参数 phone

  • 请求方式 POST

  • 返回结果
    { "msg": "获取验证码成功", "code": 200, "data": null }

3. 验证码登录
  • URL: http://121.4.51.120/ferry/user/loginByCode

  • 请求参数 phone code

    暂未接入短信平台 验证码用0123

  • 请求方式 POST

  • 返回结果
    { "msg": "登录成功", "code": 200, "data": { "userName": "_13581505849", "userId": "4", "avatar": null, "sex": 0, "birth": 1607908580472, "phone": "13581505849" } }

4. 更新个人信息
  • URL: http://121.4.51.120/ferry/user/updateUserInfo

  • 请求参数

    userId 必需
    userName 可选
    sex 可选 birth 可选 时间戳 file 可选 要上传的头像

  • 请求方式 POST

  • 返回结果

    字段解释之后在加

{
    "msg": "更新成功",
    "code": 200,
    "data": {
        "userName": "test001",
        "userId": "4",
        "avatar": null,
        "sex": 1,
        "birth": 1607268615522,
        "phone": "13581505849"
    }
}

Platform 模块

1. 获取平台
  • URL: http://121.4.51.120/ferry/platform/getPlatform?userId={userId}
  • 请求方式 GET

  • 返回结果

    说明 type 0 信用卡 1 三方平台 2 其他 

 {"msg": "成功", "code": 200, "data": {"creditCardList": [ { "id": 1, "name": "光大银行", "type": 0, "icon": null,"userId": "0" } ],"tripartiteList": [{"id": 11,"name": "baibaile","type": 2,"icon": null,"userId": "1"}]}}
2. 添加平台
  • URL: http://121.4.51.120/ferry/platform/addPlatform

  • 请求方式 POST

    header: content-type application/json
    参数格式json 例子： {"name":"XXX", "type":1,"icon":"","userId":1}

  • 返回结果 

 {
    "msg": "添加成功",
    "code": 200,
    "data": {
        "id": 13,
        "name": "XXX",
        "type": 1,
        "icon": "",
        "userId": "1"
    }
}
2. 删除平台
  • URL: http://121.4.51.120/ferry/platform/addPlatform
  • 请求方式 POST
  • 参数 id 要删除平台的id
  • 返回结果
{
    "msg": "删除成功",
    "code": 200,
    "data": {
        "id": 11,
        "name": "baibaile",
        "type": 2,
        "icon": null,
        "userId": "1"
    }
}
